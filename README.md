# engenharia-de-software

Repositório utilizado para concentrar as implementações de um projeto da matéria Engenharia de Software, onde foi proposta a elaboração de um sistema Web simples utilizando Flask.

A arquitetura do projeto foi inspirada num projeto passado como exemplo em aula e adaptado.

O sistema em desenvolvimento serve para fazer o registro de vendas de um supermercado, envolvendo
- Produtos
- Clientes
- Estoque
- Vendas
